     30== Log Format == 
     31 
     32Starting with Trac 0.10.4 (see [trac:#2844 #2844]), it is possible to set the output format for log entries. This can be done through the `log_format` option in [wiki:TracIni#logging-section trac.ini]. The format is a string which can contain any of the [http://docs.python.org/lib/node422.html Python logging Formatter variables]. Additonally, the following Trac-specific variables can be used: 
     33 '''$(basename)s''':: The last path component of the current environment. 
     34 '''$(path)s''':: The absolute path for the current environment. 
     35 '''$(project)s''':: The originating project's name. 
     36 
     37Note that variables are identified using a dollar sign (`$(...)s`) instead of percent sign (`%(...)s`). 
     38

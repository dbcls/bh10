For the SemWeb? installation contest I was looking on the web for a simple way to install Virtuoso on my virtualized Ubuntu server 9.10 It seems that kubuntu guys are very excited about semantic web & desktop  http://www.kubuntu.org/news/kde-sc-4.4-rc-2 and their interest come from this application  http://nepomuk.semanticdesktop.org/xwiki/bin/view/Main1/.

just few lines: 

sudo aptitude install python-software-properties
sudo add-apt-repository ppa:kubuntu-ppa/beta
sudo aptitude install virtuoso-server
sudo aptitude install virtuoso-drivers
